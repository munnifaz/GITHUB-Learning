## What is a Fork?

A **fork** is your personal copy of someone else's repository. It allows you to:

- Freely experiment with changes
- Develop features or fix bugs
- Propose improvements via **Pull Requests**

---

## How to Use This Repository

1. **Fork** this repository using the `Fork` button in the top-right corner.
2. **Clone** your forked copy:
   ```bash
   git clone https://github.com/Houssam-nxy/Spoon-Knife.git
   cd Spoon-Knife
   ```
3. **Make some changes** ‚Äî edit the HTML, add your name, tweak the style, etc.
4. **Push your changes**:
   ```bash
   git add .
   git commit -m "My changes for practice"
   git push origin main
   ```
5. **Submit a Pull Request** to share your changes:  
      [Open a Pull Request](https://github.com/octocat/Spoon-Knife/pulls)
